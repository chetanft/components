# InputNumber Contract

## Path
- `src/components/molecules/InputNumber`

## API
- Sizes: `sm`, `md`, `lg` (via `ComponentSize`)
- Controls positions: `right`, `both`
- Props: `value`, `defaultValue`, `min`, `max`, `step`, `precision`, `controls` (boolean), `controlsPosition`, `error` (boolean), `disabled`, `formatter`, `parser`
- Composable sub-components: `InputNumberWrapper`, `InputNumberField`, `InputNumberPrefix`, `InputNumberSuffix`, `InputNumberButton`, `InputNumberControls`

## Token Rules
- No hex literals in component styles.
- Spacing and dimensions must resolve through tokens/classes.
