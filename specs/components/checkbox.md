# Checkbox Contract

## Path
- `src/components/atoms/Checkbox`

## API
- Sizes: `sm`, `md`
- States: `checked`, `unchecked`, `indeterminate`
- Sub-components: `CheckboxInput`, `CheckboxLabel`, `CheckboxHelper`, `CheckboxError` (composable API)

## Token Rules
- No hex literals in component styles.
- Spacing and dimensions must resolve through tokens/classes.
