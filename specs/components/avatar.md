# Avatar Contract

## Path
- `src/components/atoms/Avatar`

## API
- Sizes: `xxs`, `xs`, `sm`, `md`, `lg`, `xl`, `xxl` (or numeric px value)
- Shapes: `circle`, `square`
- Sub-components: `AvatarImage`, `AvatarFallback` (composable API); `AvatarGroup` (grouping)

## Token Rules
- No hex literals in component styles.
- Spacing and dimensions must resolve through tokens/classes.
