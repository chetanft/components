# Alert Contract

## Path
- `src/components/molecules/Alert`

## API
- Variants: `info`, `success`, `warning`, `danger`
- Radius: `none`, `sm`, `md`, `lg`
- Sub-components: `AlertIcon`, `AlertTitle`, `AlertDescription`, `AlertAction`, `AlertClose` (composable API)

## Token Rules
- No hex literals in component styles.
- Spacing and dimensions must resolve through tokens/classes.
