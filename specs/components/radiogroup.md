# RadioGroup Contract

## Path
- `src/components/atoms/RadioGroup`

## API
- Sizes: `sm`, `md`
- Orientations: `horizontal`, `vertical`
- Props: `name` (required), `value`, `defaultValue`, `onChange` / `onValueChange`, `disabled`
- Composable sub-components: `RadioItem`, `RadioItemInput`, `RadioItemLabel`, `RadioGroupLabel`, `RadioGroupHelper`, `RadioGroupError`

## Token Rules
- No hex literals in component styles.
- Spacing and dimensions must resolve through tokens/classes.
