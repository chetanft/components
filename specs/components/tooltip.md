# Tooltip Contract

## Path
- `src/components/molecules/Tooltip`

## API
- Placement: `top`, `bottom`, `left`, `right`
- Alignment: `start`, `center`, `end`
- Colors: `white`, `dark`
- Sub-components: `TooltipTrigger`, `TooltipContent`, `TooltipTitle`, `TooltipDescription`, `TooltipArrow` (composable API)

## Token Rules
- No hex literals in component styles.
- Spacing and dimensions must resolve through tokens/classes.
