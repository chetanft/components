# Statistic Contract

## Path
- `src/components/atoms/Statistic`

## API
- Label placements: `Below` (default), `Top`
- Props: `labelPlacement`
- Composable sub-components: `StatisticValue`, `StatisticTitle`
- Note: `labelPlacement="Top"` reverses the child render order internally.

## Token Rules
- No hex literals in component styles.
- Spacing and dimensions must resolve through tokens/classes.
