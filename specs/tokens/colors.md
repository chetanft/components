# Token Spec: Colors

## Rule
- Use semantic color variables/classes (`--color-*`, Tailwind semantic classes).
- Hex and rgba literals in component source are violations unless documented.

## Primary Semantic Tokens
- `--color-primary`
- `--color-secondary`
- `--color-tertiary`
- `--color-bg-primary`
- `--color-bg-secondary`
- `--color-border-primary`
- `--color-border-secondary`
- `--color-neutral`, `--color-positive`, `--color-warning`, `--color-critical`
