# Next.js Pages Router Template with FT Design System

This template provides a pre-configured Next.js Pages Router project with FT Design System.

## Setup

1. Copy files to your project directory
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run development server:
   ```bash
   npm run dev
   ```

## Configuration

- **CSS Import**: Already added in `pages/_app.tsx`
- **Tailwind Config**: Pre-configured with FT DS content paths
- **Components**: Example usage in `pages/index.tsx`

## Next Steps

- Import components: `import { Button, Input } from 'ft-design-system';`
- Customize `pages/index.tsx` with your content
- Add more pages in the `pages` directory

For more information, see [FT Design System Documentation](../../docs/INTEGRATION_GUIDE.md).
